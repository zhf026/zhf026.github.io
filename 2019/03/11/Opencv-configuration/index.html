如何在Python中配置opencv

发表于 2019-03-11 | 更新于 2019-03-13 | 分类于 Opencv

   讲解如何在Python中安装opencv库（Windows环境）

   之前在使用opencv库（版本号4.0.0）时，发现有些模块或函数不能成功调用，比如drawKeypoints这个函数。经过一系列的Google和Baidu之后，成功地解决了问题。
   opencv库有两种，一种是opencv-python，另一种是opencv-contrib-python。前者是基础库，后者则是在前者基础上加了一些contrib module（视觉处理一般用这个）。
   在这里，我们使用版本号为3.4.2.16的opencv库。

1. 卸载（如果之前没安装过，请忽略）

pip uninstall opencv-python或pip uninstall opencv-contrib-python

2. 安装

pip install opencv-python==3.4.2.16或pip install opencv-contrib-python==3.4.2.16
   一个快速安装库的方案：先下载好.whl文件，在cmd里输入pip isntall，再将下好的文件(.whl)拖入cmd窗口即可。
