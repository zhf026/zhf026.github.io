四. 链接到 Github

   首先你得有个Github账号，没有的话，盘它！有的话，直接继续下面：

  1. 创建repo（仓库）
       在你的Github中创建一个repo，并命令为yourname.github.io，其中yourname是你github的名称。在你的Github中创建一个repo，并命令为yourname.github.io，其中yourname是你github的名称。
  2. 配置Github账号信息（在Git Bash中操作）
      1）创建.ssh文件夹：输入命令mkdir ~/.ssh
      2）进入ssh文件夹：输入命令cd ~/.ssh
      3）输入命令git config --global user.name "your_github_name"和git config --global user.email "your_email@example.com"，其中邮箱为你注册Github时用的邮箱。
  3. 创建SSH（在Git Bash中操作）
       在git bash中输入命令ssh-keygen -t rsa -C "youremail@example.com，其中邮箱为你注册Github时用的邮箱。
  4. ssh key
       找到.ssh文件夹，在文件内找到id_rsa.pub，将文件后缀名改为.txt，用记事本打开，复制文件内容。
  5. 在Github中设置SSH
       登陆到Github，进入Settings，选择SSH and GPG Keys，再选择New SSH Key，将你复制的内容粘贴即可。

八. 基本操作

  1. 打开Git Bash；
  2. 输入命令cd F:\\ZHF\\myblog进入myblog文件夹下；
